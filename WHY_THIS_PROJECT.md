# WHY THIS PROJECT

### Overview

This project's main purpose and focus is to help me understand and develop my skills in the software engineering life cycle.
This project will show:

- My learning experience and process
- The way certain technologies were used
- How I struggled with each one
- How I used AI to help me learn

This project is in no way perfect and basically acts a playground for me to test the limits of the technologies used and help my understanding of them.

### Why This Project

After successfully completing my internship, I was exposed to a number of technologies and software which I would have never been exposed to before.
Recreating Block Blast would allow me to develop my problem-solving skills, critical thinking skills and self learning skills.

The technologies used in this project are:

- Python
- React
- Typescript
- FastAPI
- Docker
- Kubernetes (briefly)

I have seen these technologies used during my internship but only mainly used Python.
Using these technologies has helped me develop a deeper understanding of

- How everything in the code links together
- How difficult it is to build software
- How difficult it is to design software architecture
- How useful AI is for learning

### How Was AI Used

In this project, AI was used as a way to develop the frontend, teach me why certain technologies were used and how to use it.
AI was also used in the K8 part. However, the K8 part is more of an addon since I wasn't really mean to do it I just wanted to learn it by using minikube. Docker compose is really all I need for this project.
